# Щукин — материалы (полный корпус, скопировано по явному запросу)

Щукин = Дмитрий Щукин, курс «Криптограмотность». Забрано с ветки
`claude/crypto-video-transcripts-PnCMQ` (коммит `9d7c54e`), куда
транскрипты залила параллельная сессия.

Это **сырьё / справка, не правила**. По правилам проекта чужой метод идёт
путём: идея → формализация → бэктест → если edge подтверждён, встраиваем.
Прямо в правила или live ничего отсюда не переносится.

## Состав

- `youtube/` — 100 транскриптов YouTube-канала (`001`–`100`) + `INDEX.md`.
- `курс/уроки/` — 32 урока курса + `INDEX.md`.
- `курс/vip-встречи/` — 8 записей VIP-встреч.
- `курс/конспект/` — структурный разбор (основные идеи, правила входа/выхода,
  риск-менеджмент, что-проверить, что-отбросить).
- `курс/триаж.md` — что формализуемо, приоритет (P0: видео 008/005/009).
- `курс/каталог-длинных-видео.md` — каталог 14 длинных видео.
- `курс/README.md`, `курс/HANDOFF.md` — статус и хендофф по курсу.
- `_канал-описание.md` — описание канала.

## ⛔ DOLF — исключено по требованию пользователя

**DOLF** — фирменная стратегия Щукина: композит из **D** (CVD, дельта
объёма), **O** (Open Interest), **L** (Liquidations), **F** (Funding).
Сборник DOLF-сетапов: `raw/DOLF-сетапы.md` в исходной ветке
(teletype.in/@cryptogramotnost/DOLF_setup).

Пользователь явно сказал: материалы Щукина берём, но **стратегию DOLF
не применяем**. Поэтому:

- Файл `DOLF-сетапы.md` сюда **намеренно не скопирован**.
- DOLF **не формализуем, не бэктестим, не встраиваем** в стратегию/правила
  бота «трейдер».
- В транскриптах уроков/VIP концепции CVD/OI/ликвидаций/фандинга всё равно
  упоминаются как сырьё — это нормально (справка), но **сборка их в
  DOLF-сетапы под запретом**. Если из этих метрик когда-либо строим что-то —
  это должна быть своя, независимая от DOLF логика, с отдельным планом.

Оригинал DOLF-сетапов при необходимости: ветка
`claude/crypto-video-transcripts-PnCMQ`,
`бизнес/материалы/курс-криптограмотность/raw/DOLF-сетапы.md`.
