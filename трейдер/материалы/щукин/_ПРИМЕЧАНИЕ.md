# Щукин — материалы (полный корпус, скопировано по явному запросу)

Щукин = Дмитрий Щукин, курс «Криптограмотность». Забрано с ветки
`claude/crypto-video-transcripts-PnCMQ` (коммит `9d7c54e`), куда
транскрипты залила параллельная сессия.

Это **сырьё / справка, не правила**. По правилам проекта чужой метод идёт
путём: идея → формализация → бэктест → если edge подтверждён, встраиваем.
Прямо в правила или live ничего отсюда не переносится.

## Состав

- `youtube/` — 100 транскриптов YouTube-канала (`001`–`100`) + `INDEX.md`.
- `курс/уроки/` — 32 урока курса + `INDEX.md`.
- `курс/vip-встречи/` — 8 записей VIP-встреч.
- `курс/конспект/` — структурный разбор (основные идеи, правила входа/выхода,
  риск-менеджмент, что-проверить, что-отбросить).
- `курс/триаж.md` — что формализуемо, приоритет (P0: видео 008/005/009).
- `курс/каталог-длинных-видео.md` — каталог 14 длинных видео.
- `курс/README.md`, `курс/HANDOFF.md` — статус и хендофф по курсу.
- `_канал-описание.md` — описание канала.

## Композитная стратегия — применяем, фирменное слово — нет

Композитная стратегия Щукина = **CVD + Open Interest + ликвидации +
фандинг**. Её **берём в работу** (формализация → бэктест → встройка по
стандартному пути проекта).

Ограничение только терминологическое: **фирменный 4-буквенный бренд-акроним
этой стратегии в проекте не используется** — ни в коде, ни в конфигах, ни в
именах файлов, ни в планах/правилах. Везде нейтральное название «композит
CVD·OI·ликвидации·фандинг».

- Сборник сетапов скопирован сюда под нейтральным именем:
  `курс/сетапы-композит.md` (фирменное слово в тексте заменено на
  нейтральное, содержание сетапов сохранено).
- Оригинал (с фирменным термином) при необходимости: ветка
  `claude/crypto-video-transcripts-PnCMQ`, файл сборника сетапов в
  `бизнес/материалы/курс-криптограмотность/raw/`.
