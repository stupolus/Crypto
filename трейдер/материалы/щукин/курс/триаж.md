# Триаж курса → что формализуемо

## Дата: 2026-05-15 · План: plans/20-стратегия-из-курса.md

## Честная оценка источника

Это **YouTube-канал** (кликбейт-заголовки «разгоняем депозит 1000$»,
воронка в телеграм-сигналы), не структурированный курс с PDF. Заголовки
правил НЕ содержат — нужны транскрипты. Часть видео — мотивация/промо
(014 — бонус ByBit, 010 — реклама ботов, 079/082/085/086/089 — «разбор
сделки на 700$» = маркетинг).

**НО**: повторяющиеся темы канала совпадают с тем, что в проекте УЖЕ
есть как примитивы. Это хороший знак — методология «торговля от
ликвидаций + открытый интерес + дельта объёма» формализуема.

## Темы канала → существующие сигналы проекта

| Тема канала (видео) | Примитив в проекте |
|---------------------|--------------------|
| Торговля от ликвидаций (008, 040, 042, 090, 091) | `core/signals/liquidation_sweep.py` |
| Резкий рост Open Interest из боковика = LONG (076, 088) | `core/signals/funding_extreme.py` + OI (расширить) |
| Дельта объёма / CVD (011, 067, 093) | `core/signals/order_flow.py` (compute_imbalance) |
| Боковые объёмы / уровни-плотности (031, 037, 095) | volume profile (нет — кандидат на новый сигнал) |
| Heatmap ликвидаций (092) | Coinglass adapter (план, не реализован) |
| Фибоначчи-уровни входа (027, 050) | нет (кандидат) |
| Мульти-тейк фиксация (028, 045, 049) | TP-логика в strategy (расширить) |
| Риск-таблица / позиционирование (013) | `core/risk` (уже есть, сверить) |
| Стоп за ликвидность, не выбивать (033, 072, 094) | SL-логика (сверить с liquidation_sweep) |

Вывод: канал описывает **composite-сигнал** (ликвидации + OI + CVD +
уровни), а не один триггер. Это ровно принцип CLAUDE.md §3
(«Pump+OI+Volume+Funding+Heatmap = сигнал»). Совпадение архитектуры.

## Приоритет транскрипции (от пользователя нужны транскрипты)

Не все 100 видео — только те, где конкретная механика входа/выхода:

**P0 (ядро стратегии — без них никак):**
- **008** «Торговля от ликвидаций + точки входа» (46м) — главное видео
- **005** «Стратегия в плюс на фьючерсах» (89м) — полная стратегия
- **009** «Лучшая стратегия для начинающих» (50м) — базовая система

**P1 (фильтры/подтверждения):**
- **011** «Дельта объёмов / CVD» (22м) — подтверждение входа
- **013** «Риск-таблица» (29м) — позиционирование (сверить с риск-профилем)
- **004** «ТОП-7 индикаторов» (23м) — какие индикаторы канал считает рабочими

**P2 (управление позицией):**
- **006** усреднение, **007** хеджирование — НЕ для MVP (CLAUDE.md против
  усреднения убыточных). Транскрибировать для полноты, но в стратегию
  не берём без отдельного обоснования.

Shorts 040/042/046/051/052/067/076/088 — короткие версии P0/P1, можно
использовать для уточнения если длинные нечёткие.

## Что нужно от пользователя (Phase 0 продолжение)

Транскрипты P0+P1 (6 видео). Варианты:
1. **YouTube авто-субтитры** — открыть видео → «...» → Показать
   транскрипцию → скопировать текст → прислать (по одному видео)
2. **Скачать видео** (yt-dlp) → выложить файлы → прогоню Whisper
3. Если есть **платная версия курса** отдельно (не YouTube) с PDF/
   структурой — она приоритетнее, заголовки тут кликбейт

Самое быстрое — вариант 1 для видео 008, 005, 009 (P0). С них начнём
формализацию: liquidation-entry + OI-breakout + CVD-confirm уже имеют
примитивы в `core/signals/`, нужно только узнать ТОЧНЫЕ правила
(какой объём ликвидаций, какой рост OI, какой timeframe, где SL/TP).
