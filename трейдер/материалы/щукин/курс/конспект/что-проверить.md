# Что проверить бэктестом (гипотезы)

⚠️ Ведро «догма без доказательств» + ✅ гипотезы для edge-проверки.
Каждая — H. Подтвердилась на 2+ годах OOS → в стратегию. Нет → отброс.

## Блокеры (без них Phase 2 невозможна)

- **B1. Нет фида открытого интереса.** Вся методология построена на
  OI как главном gate (шорт только на падающем OI и т.п.). В проекте
  OI-сигнала нет. Нужно: источник OI (BingX/Coinglass) + примитив
  `core/signals/open_interest.py`. БЕЗ ЭТОГО формализовать нельзя —
  только liquidation+CVD часть, что половина логики.
- **B2. Исторические ликвидации.** liquidation_sweep работает на live
  данных Coinglass; для бэктеста нужна история ликвидаций (есть ли у
  Coinglass экспорт за 2 года?). Если нет — проксировать через
  большие knot-свечи объёма (грубо, отметить как приближение).

## Гипотезы стратегии

- **H1.** Liquidation Reversal (сетап A1/A2) на BTC/ETH/SOL даёт
  PF > 1.3, Sharpe > 0.8, ≥30 сделок OOS на 2+ годах. (главная)
- **H2.** OI-breakout из боковика (сетап B) опережает цену и ловит
  >50% движения (заявление видео 076). Проверить: средний MFE после
  сигнала vs random entry.
- **H3.** Фильтр «фандинг ≤ −1.5% → не шортить» уменьшает убыток
  шортов (видео 004 «фаза дойки»). A/B: стратегия с фильтром vs без.
- **H4.** Порог ликвидации $100k для мажоров действительно маркирует
  локальные развороты (видео 012, выборка 3-4 примера — мало).
- **H5.** CVD-price дивергенция на ключевых уровнях > CVD-дивергенция
  «посреди графика» (видео 011 утверждает, нужна статистика win-rate
  по локации).

## Догмы курса (требуют беспристрастной проверки)

- «80% времени рынок в боковике, 20% тренд» — измерить на BTC 2+ года
  по выбранному TF/определению боковика.
- «Цена никогда не вернётся в зону огромных объёмов» — «никогда»
  фальсифицируемо: доля возвратов в Volume POC за период.
- «Single-ликвидация >$100k всегда даёт локальный разворот» — частота
  разворота vs продолжения после таких событий.
- «Высокий негативный фандинг → продолжение пампа почти гарантировано»
  — распределение forward-return при фандинге < порога.
- «1:3 при 33% винрейт = безубыток» — арифметика верна, но винрейт
  стратегии A1/A2 неизвестен; именно его и меряем (H1).
