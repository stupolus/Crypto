# Правила выхода

✅/🟡. Из видео 008, 005, 013, 011.

## Тейк-профит

- **R:R стартовое 1:3** (риск 0.4-0.5% ради 1.5%). Запрещён 1:1
  («казино»). Позже допустимо 1:2.
- Стиль активной сессии (видео 005): короткие тейки **2-6%**,
  отработка 10-15 мин. Это скальп-режим — для нашего бота скорее
  反пример (CLAUDE.md против скальпинга), берём 1:3 swing-вариант.
- Лонг закрывать при развороте тренда: CVD начинает снижаться +
  OI снижается (поздние лонги заперты) → выход.
- Шорт закрывать симметрично: CVD/OI разворот вверх.

## Стоп-лосс

- За ближайший значимый минимум/максимум (или рядом с равными
  экстремумами). Не «фиксированный %», а структурный.
- Зависимость SL от таймфрейма (видео 013):
  - 5m → SL ≈ 0.4%
  - 30m → SL ≈ 1.2%
  - 4h → SL ≈ 4.4%
- Размер позиции = риск / дистанция стопа (см. риск-менеджмент.md).
- Допустим re-entry после стоп-аута если зона/ликвидация всё ещё
  валидна (split-вход).

## Дисциплина исполнения

- Поставил SL+TP — терминал НЕ трогать до закрытия сделки.
- Не закрывать руками «в минимальный плюс» вместо стопа (в видео 005
  автор сам это нарушил на KNC — отметка в что-проверить.md).

## Для кодификации

Выход в стратегии = attached SL/TP при входе (как в btc_breakout):
- SL: структурный (за экстремум окна) ИЛИ min-distance по риск-профилю
- TP1: entry ± 3R (1:3) — НЕ 1.5R как в текущем btc_breakout
- Trailing после TP1 — отдельная итерация (как в плане 08)
