# Правила торговли — бот «трейдер»

Свои правила этого бота. Отличаются от основного бота. Цифры риска — в
`риск-профиль.md` (не дублируем здесь).

> Статус: рабочая версия v1.

## Инструменты

- **BTC** — основа портфеля.
- **SOL** — вторая крипта.
- **Золото (XAU)**.
- **Серебро (XAG)** — торгуем особенно коротко (история: при 5x чуть не
  ликвиднуло; теперь плечо ≤ 3x и очень короткие сделки).

## Стиль и горизонт

- **Скальп**: сделки минуты — десятки минут, не перенос на ночь по умолчанию.
- Серебро — самые короткие удержания из всех.

## Входы

- Композит сигналов, не один скринер (принцип проекта).
- Конкретные триггеры по инструменту — задаются в планах (`трейдер/планы/`)
  и подтверждаются бэктестом. До бэктеста стратегия не считается готовой.
- Фильтр профит:риск ≥ 1.5R обязателен (см. `риск-профиль.md`).

## Выходы

- Стоп-лосс: биржевой, вместе с входом, на волатильностной дистанции.
- Тейк: ≥ 1.5R; частичная фиксация / перенос в безубыток — параметры в плане.
- Не высиживаем убыток без стопа. Глубже просадка — только в рамках стопа.

## Чужие сигналы

Идею перенять можно, но в live-вход — только после бэктеста/статистики.
Непроверенный чужой сигнал прямо в live — нельзя (CLAUDE.md принцип 1).

### ⛔ DOLF — не применяем (решение пользователя)

Стратегия **DOLF** Щукина (композит CVD+OI+Liquidations+Funding) —
**не формализуем, не бэктестим, не встраиваем** в этого бота. Материалы
Щукина используем как справку, саму DOLF-сборку — нет. Если из метрик
CVD/OI/ликвидаций/фандинга строим что-то, это своя независимая логика с
отдельным планом, не DOLF. Детали: `материалы/щукин/_ПРИМЕЧАНИЕ.md`.

## Оценка результатов (требование к метрикам)

Целевой винрейт **не задаём** — он не критерий сам по себе. Вместо этого
журнал и отчёты считают статистику **в разрезах**:

- по каждому инструменту отдельно (BTC / SOL / золото / серебро);
- по направлению отдельно (long vs short) внутри каждого инструмента;
- метрики на разрез: число сделок, винрейт, средний R, expectancy,
  макс. просадка, профит-фактор.

Решение «работает / не работает» — по разрезам, а не по общему винрейту.
