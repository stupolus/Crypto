# Риск-профиль — бот «трейдер»

Единственный источник истины по риску для бота из папки `трейдер/`.
Код и планы ссылаются сюда, не дублируют цифры. **Не наследует** корневой
`бизнес/риск-профиль.md`.

> Статус: рабочая версия v1 (стартовые значения для testnet). Все цифры —
> крутилки, меняются здесь, не в коде.

## Капитал

- Депозит / стартовое эквити: **$10 000**.
- Маржа: **изолированная** всегда. Кросс-маржа запрещена.

## Размер позиции (главная модель)

Размер считается **от риска, не от плеча**. Плечо — производная величина.

1. Базовый риск на сделку: **0.5% эквити = $50** (макс. убыток по стопу).
2. Дистанция стопа = от внутридневной волатильности инструмента (ATR на
   скальп-ТФ), не фиксированный %.
3. Размер позиции (notional) = риск ÷ дистанция стопа.
4. Эффективное плечо = notional ÷ эквити — **следствие**, не вход.

### Потолки эффективного плеча (страховка)

Это не цель, а жёсткий предел. Реальное плечо обычно ниже (из п.1–3).
Если волатильность требует плеча выше потолка — **сделка пропускается**.

| Инструмент | Потолок эфф. плеча | Прим. |
|---|---|---|
| BTC | ≤ 5x | основа |
| SOL | ≤ 5x | |
| Золото (XAU) | ≤ 5x | |
| Серебро (XAG) | ≤ 3x | только очень короткие сделки |

### Буфер ликвидации (жёстко)

Цена ликвидации должна быть **≥ 4× дистанции стопа** от входа. Если нет —
размер уменьшается или сделка пропускается. Стоп всегда срабатывает раньше
ликвидации с большим запасом.

## Стопы

- Биржевой стоп ставится **вместе с входом**. Позиции без стопа нет — никогда.
- Стоп не двигается против позиции. Перенос только в безубыток/в плюс.
- «Пересидеть просадку» без стопа — запрещено (это и есть механизм слива).
  Глубокая просадка допустима только в пределах изначального стопа.

## Профит:риск

- Вход только если потенциальная цель **≥ 1.5× дистанции стопа** (мин 1.5R).
  «Риск оправдан прибылью» = это правило, а не высиживание убытка.

## Одновременные позиции

- Максимум **2** одновременно.
- **Не открывать 2 коррелированные.** Группы корреляции:
  - крипта: {BTC, SOL}
  - металлы: {золото, серебро}
- Т.е. максимум одна из крипты + одна из металлов. Две из одной группы — нет.

## Циркуит-брейкеры (автоматические)

| Период | Лимит убытка (от эквити) | Действие |
|---|---|---|
| День | **-4% (-$400)** | стоп торговли до след. дня |
| Неделя | -8% | стоп до след. недели (v1, крутилка) |
| Месяц | -15% | стоп + обязательный разбор (v1, крутилка) |

## Запреты (навсегда)

- Позиция без биржевого стопа.
- Усреднение в убыток.
- Кросс-маржа.
- Плечо выше потолка по таблице.
- Чужой непроверенный сигнал прямо в live-вход (см. `правила-торговли.md`).
