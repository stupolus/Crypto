## ⚠️ Поправка 2026-05-19 (ярлыки выше частично НЕВЕРНЫ)

«Н/Д — нет данных ликвидаций» по альтам — **ошибка интерпретации**.
Данные ликвидаций по XRP/BNB/DOGE/LINK/SOL/… ЕСТЬ (Coinglass отдаёт
1000 точек). Ноль сделок = к альтам применён BTC-порог
`liq_min_baseline_usd=100000` (для альтов по методике $15–30k) +
короткое 6-мес окно + OI-гейт тогда был выключен из-за неверного
эндпоинта (уже починен, план §6c). Это конфиг/мало данных, не
отсутствие фида.

Перегон не делаю: тариф жёстко ограничивает 4h-историю ~6 месяцами
(API: earliest start_time = 2025-11-20). На таком окне любой символ
даёт 1-значное число сделок (BTC с OI = 4, без = 9) — это шум,
перегон по списку = data-dredging. Нужен тариф с глубиной 4h ≥2 года
либо forward-paper-сбор. Детали — план §6c.
