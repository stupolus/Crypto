# трейдер/ — отдельный бот

Самостоятельный торговый бот. **Независим от основного бота** (`strategies/`, `core/`,
`runners/`, `бизнес/`).

## Принципы этой папки

1. **Свои правила торговли.** Бот отличается от того, что разрабатывается в другой
   сессии. Правила пишутся здесь с нуля — см. `правила/`. Старые md из `бизнес/`
   **не наследуются и не копируются**.
2. **Материалы — только по явному запросу.** `материалы/` пустая по умолчанию.
   Сюда копируется только то, что пользователь попросил явно (конкретный
   курс/канал/скринер/транскрипт). Авто-импорта и ссылок на общий `бизнес/` нет.
3. **План перед кодом.** Любая стратегия/фича — сначала файл в `планы/`,
   потом код. Стратегия без бэктеста не считается реализованной.

## Структура

- `правила/` — свои риск-профиль, правила торговли, цели (единственный источник
  истины для этого бота).
- `материалы/` — материалы, скопированные сюда по явному запросу.
- `планы/` — техпланы этого бота, один файл = одна фича.
- `стратегии/` — код стратегий этого бота.
- `ядро/` — инфраструктура (риск/сигналы/бэктест) этого бота.
- `раннеры/` — точки входа/запуск.
- `ретро/` — ретроспективы сессий по этому боту.

## Безопасность

Те же жёсткие рамки, что и в корневом `CLAUDE.md`: live только после плана +
testnet + явного «да», ключи через env, стоп ставится вместе с входом.
