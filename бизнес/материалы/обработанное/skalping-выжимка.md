## ✅ Тестируемое (кандидаты в бэктест edge_hybrid, план 33)

1. **#008 volatility-box breakout + volume-bias** — построить бокс
   консолидации по нормированной волатильности (длина норм. 20–50
   на 15m). Лонг только если объём внутри бокса нетто-бычий И свеча
   ЗАКРЫВАЕТСЯ (не фитиль) за верх бокса; стоп под середину/дальний
   край; TP RR 1.5–2. **Структурно ДРУГОЙ триггер (пробой, не
   reversion)** → должен размазать сделки по большему числу недель
   = прямое лекарство от диагноза 33.1 (кучкование). Высокий
   приоритет.
2. **#006 200-EMA режим-гейт** — лонг только price>200EMA, шорт
   только price<200EMA (отличается от текущего EMA-spread фильтра;
   может изменить дисперсию сделок).
3. **#006 UT Bot (ATR p=2, key=3)** — альтернативный триггер-лег
   с подтверждением объёмом.
4. **#003 ликвид-фильтр** — торговать только символы >$150M
   24h-объёма; биас к кластерам S/R у круглых чисел.
5. **#004 айсберг-вход** — отскок от лимита ≥10–50× медианной
   заявки. Требует L2-истории → data-gated, отложить.
6. **#007 Щукин: фейд экстремума** — рост OI + сильный buy-имбаланс
   + всплеск противоположных ликвидаций → разворот. Falsifiable,
   если есть footprint/OI/liq-серии (Coinglass) — питание
   DOLF-фильтра фазы 33.2. Funding-guard в риск-движок.

## 🟡 Риск/исполнение (в риск-движок)

- #001: в низкой волатильности/каше — НЕ форсировать сделки, ждать
  или ниже ТФ. Прямо релевантно диагнозу 33.1 (редкие сделки —
  норма, не баг; не подгонять под частоту).
- #004/#006: тугой стоп, длинный TP, частичные выходы; ширина
  стопа по ATR.
- #007: не шортить в глубоко-отрицательный funding без хеджа;
  терпение/селективность > частота (эхо #008).
