# Что отбросить (и почему)

❌ Ведро «маркетинг/мотивация» + явно вредные для нашего подхода.

## Маркетинг / мотивация — игнорируем

- «Разгони депозит с $1000», «37.5% за 6 часов», «100% винрейт
  19/19» — продуктовая риторика, не стратегия.
- Реклама VIP/Telegram-группы и собственных ботов (OI/фандинг/
  ликвидации/объём-алерты) — мы строим свои примитивы.
- Бонус ByBit $800 (видео 014), призывы лайк/подписка.
- Эмоциональные пассажи (Ferrari/Дубай, теория игр Нэша, «слона по
  кусочку», аналогия с иероглифами) — нулевая тестируемая ценность.
- Общая риторика «теханализ/Smart Money/Фибо не работают» — мнение
  без тестируемого содержания (но сам ВЫВОД — фокус на потоке
  ордеров — мы берём, см. основные-идеи).

## Вредно для нашего подхода — НЕ кодировать

- **Усреднение убыточной позиции** (видео 006) — прямо против
  CLAUDE.md (анти-скальпинг, риск от % эквити) и риск-профиля.
  Маскирует риск, ломает расчёт позиции.
- **Хеджирование убыточной «вытащить любую сделку»** (видео 007) —
  отложено; для MVP опасно (двойная экспозиция, ломает RiskEngine).
- **Скальп 2-6% тейки, отработка 10-15 мин** (видео 005 live-сессия)
  — против CLAUDE.md «не использовать HF-паттерны на интрадей,
  латенси убьёт». Берём только swing-вариант (1:3, structural SL).
- **Ручное закрытие «в минимальный плюс» вместо стопа** — автор сам
  это сделал на KNC (видео 005), нарушив свои правила. Бот ДОЛЖЕН
  держать стоп; это анти-пример дисциплины.

## Нерелевантно технически

- Видео 010 (реклама ботов), 014 (бонус) — нулевая стратегия.
- Видео 088 — 38-сек выжимка OI, дублирует 012/076, новых правил нет.
- 86 shorts — сжатые повторы длинных, использовать только для
  уточнения если длинное видео нечёткое.

## Резюме

Берём: liquidation-reversal + OI-breakout + CVD/funding фильтры +
1:3 R:R + structural SL + риск 1%.
Отбрасываем: усреднение, хедж-убыточной, скальп-HF, ручные правки
позиции, весь маркетинг.
