# Никита Ануфриев | Хедлайнеры (TG)

## Описание

TG-канал с **макро-структурными наблюдениями** по крипте: рыночные доли бирж, аномалии funding rate, RWA-перпы, нестандартные эпизоды.

## Тип контента

- ✅ Макро-аналитика (доли рынков, OI на биржах, RWA-сегмент)
- ✅ Аномалии funding rate
- ✅ Структурные сдвиги (Hyperliquid, HIP-3, RWA-биржи)
- ❌ Не торговые сигналы («лонг тут, цель там») — таких в канале мало
- ⚠️ Иногда «инсайдерские» тезисы по геополитике — **игнорируем**

## Как используем

| Что в канале | Что делаем |
|---|---|
| Структурные наблюдения по бирже | Записываем в `материалы/каналы/headliners-выжимки.md` (создавать при появлении) |
| Аномалия funding на конкретном инструменте | Сверяем с собственным funding-парсером, если порог сработал — фиксируем в `data/funding/events/` |
| Геополитические тезисы | Не торгуем по ним. Возможно используем как новостной контекст |
| Доли бирж по сегментам | Пересматриваем выбор venue (например, RWA-перпы — Hyperliquid, не BingX) |

## Известные сильные тезисы

### RWA-перпы: Hyperliquid доминирует
По данным канала (24.04.26, ссылка на loris.tools):
- Hyperliquid (через tradexyz, HIP-3) — ~52% мирового рынка RWA-перпов
- Binance — 13.8%
- Bybit — 5.5%
- BingX — 0.2%

**Вывод для нас:** для золота, индексов, акций ликвидность на BingX **тонкая**. В фазе 2 нужен отдельный аудит ликвидности vs Hyperliquid/tradexyz.

### Funding extremes как сигнал
Пример из канала (03.05.26): CLUSDT (нефть) на Binance, funding −0.5% / 4ч = 1095% годовых для шортов. **Не торговать по тезису канала, но использовать сам паттерн extreme funding как фичу.**

## Ритм проверки

- Раз в неделю — пробежка по каналу.
- Что-то структурное → выжимка в `материалы/выжимки/<дата>-<тема>.md`.
- Что-то измеримое (funding-аномалия) → проверка собственным парсером.

## Анти-паттерны

- Не покупать «инсайдерские» тезисы.
- Не торговать в момент публикации (если тезис верный, его уже отыграли).
- Не подменять собственный анализ мнением канала.
