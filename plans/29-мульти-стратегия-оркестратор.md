# План 29 — Оркестратор нескольких стратегий (архитектура)

## Дата: 2026-05-15 · Родитель: plans/19 (multi-runner), CLAUDE.md риск

## Запрос пользователя

Несколько стратегий запущены одновременно, возможно на одних
монетах; volume_momentum и интрадей — «живут отдельно».

## Жёсткая предпосылка (честно)

**Оркестратор строится ТОЛЬКО после того как ≥1 стратегия
прошла честный гейт** (PF>1.3 И Sharpe>0.8 И ≥30 OOS И
walk-forward И бьёт B&H). Сейчас прошедших — НОЛЬ. Запуск
нескольких no-edge стратегий = ×N способов терять + двойные
комиссии/funding, НЕ диверсификация. Это план-заготовка, НЕ
сигнал запускать сейчас.

## Проблемы, которые оркестратор ОБЯЗАН решать

1. **Нетинг по инструменту.** CLAUDE.md: ≤1 позиция/инструмент.
   A лонг BTC + B шорт BTC → нетто-позиция, НЕ две; иначе
   двойные издержки/funding на нулевой экспозиции.
2. **Портфельная агрегация риска.** Дневной −3% / нед −7% /
   мес −15% — лимиты на ВЕСЬ счёт, не на стратегию. Стоп-день
   гасит все стратегии разом.
3. **Аллокация капитала** между стратегиями (равно / по
   обратной волатильности / по доказанному Sharpe). Сумма
   риск-бюджетов ≤ общий лимит.
4. **Корреляция экспозиции.** Доказано: крипта ≈ 1 ставка.
   Оркестратор считает СОВОКУПНУЮ направленную экспозицию по
   коррелированным активам, не по символам наивно.
5. **Разрешение конфликтов** сигналов на один инструмент
   (приоритет по доказанному edge / по таймфрейму / нетто-вес).
6. **Изоляция отказа** одной стратегии (исключение в одной не
   роняет раннер и не блокирует риск-гейты остальных).

## Архитектура (эскиз)

- `StrategyRunner` (есть) на стратегию → даёт намерения (intents).
- `PortfolioOrchestrator`:
  - собирает intents всех стратегий за тик;
  - агрегирует по инструменту (нетто-цель позиции);
  - проверяет ПОРТФЕЛЬНЫЙ RiskEngine (общий эквити, лимиты,
    корреляц. экспозиция);
  - аллоцирует риск-бюджет; формирует итоговые ордера;
  - один источник правды по позициям/PnL.
- Стратегии остаются изолированными модулями (volume_momentum,
  интрадей и пр. «живут отдельно» как и просил пользователь —
  отдельные модули, общий оркестратор сверху).

- Портфельные лимиты CLAUDE.md — неприкосновенны.
- Без live без 4 нед демо + явного подтверждения.
- Интрадей на 5/15/30m как скальп — запрещён (анти-скальпинг.md);
