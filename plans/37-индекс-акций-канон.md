# План 37 — Индекс акций: канонические стратегии на 40+ лет

## Контекст (почему это НЕ p-hacking)

Биржевой индекс (S&P 500 ^GSPC с 1970, ~56 лет; NASDAQ-100
^NDX с 1985, ~40 лет, Yahoo) — единственный класс, где edge
реален и доказан десятилетиями out-of-sample: **премия за
риск акций** (рынок растёт век+) + трендовый риск-менеджмент.
Тестируем НЕ перебор, а 3 ПРЕДЗАДАННЫЕ канонические стратегии
из литературы. Это отвечает запросу владельца «индекс, 40 лет,
часы/дни/недели/полгода/год» честно.

## Честные границы

- «Часы»: 40 лет часовых данных не существует; внутридневного
  систематического edge на индексе нет (та же причина, что
  скальп). Это фиксируем, не выдумываем.
- Реальный горизонт edge индекса = месяцы–годы (премия за риск
  + тренд). Дни/недели — производно (та же дневная модель).

## Стратегии (зафиксированы из литературы, НЕ скан)

1. **Buy&Hold** — базовая премия за риск (сам по себе edge).
2. **Faber 200-дн (10-мес) SMA-тайминг** (Faber 2007): в лонге
   если close > SMA200, иначе кэш. Канон, десятилетия OOS.
3. **12-мес TSMOM**: лонг если ret_12м>0, иначе кэш.
Ребаланс — на закрытии при смене сигнала. Без шортов (для
лонг-only индекса канон; шорт индекса против премии — априори
проигрыш). Кост на переключение 0.05%.

## Гейт / робастность (честно)

Метрики: CAGR, ann.vol, Sharpe, maxDD, %+ лет. Ключ —
**по-декадная робастность** (1970s/80s/90s/00s/10s/20s):
декады = независимые режимы (включая 2000, 2008, 2020, 2022).
Стратегия «работает» ⇔ положительна в БОЛЬШИНСТВЕ декад и
тайминг снижает maxDD при сопоставимом CAGR. Это, а не один
Sharpe, и есть честный вывод.

## 10 причин провала / оговорок

1. ^GSPC = price index без дивидендов → CAGR занижен (≈−2%/г).
2. Faber-тайминг отстаёт в сильный бык (кэш на откатах).
3. Whipsaw в боковике (частые ложные переключения).
4. 2 индекса × 3 стратегии — мало (канон, не скан) но multi-
   testing держим в уме.
5. Прошлые 40 лет = вековой бык США; будущее не гарантировано.
6. BingX-перп на индекс ≠ сам индекс (издержки/funding/гэпы).
7. Налоги/funding перпа не моделируются.
8. «Часы» — данных и edge нет, честно говорим.
9. Кост 0.05%/переключение упрощён.
10. Бэктест ≠ деньги: TradFi-перп на BingX = свой риск-разговор.

## 37.3 ВЕРДИКТ (2026-05-17) — ПЕРВЫЙ робастный edge сессии

`scripts/equity_index_eval.py`, реальные данные Yahoo:

| инстр./стратегия | CAGR | Sharpe | maxDD | +декад |
|---|---|---|---|---|
| S&P500 BH | +8.1% | 0.46 | −57% | 5/6 |
| **S&P500 Faber200** | **+16.4%** | **1.40** | **−13%** | **6/6** |
| S&P500 TSMOM252 | +12.2% | 0.95 | −19% | 6/6 |
| NASDAQ100 BH | +14.1% | 0.52 | −83% | 4/5 |
| **NASDAQ100 Faber200** | **+27.0%** | **1.37** | **−32%** | **5/5** |
| NASDAQ100 TSMOM252 | +19.7% | 0.91 | −39% | 5/5 |

**Faber 200-дн SMA-тайминг — РОБАСТЕН во ВСЕХ декадах**
(1970s стагфляция / 2000 доткомы / 2008 GFC / 2020 ковид /
2022), предзадано из литературы (Faber 2007), не скан.
Sharpe 1.4, maxDD −57%→−13%. Это первое за сессию (планы
31–37), что честно проходит робаст-проверку — потому что у
индекса акций ЕСТЬ фундамент (вековая премия за риск) +
трендследование с уходом от крахов = задокументированный,
воспроизведённый феномен. У крипто-скальпа фундамента нет —
вот ключевая разница.

**Честные границы (не выдумка):**
- Суть edge = риск-менеджмент ЛОНГ-беты, не альфа сверх рынка.
- ^GSPC/^NDX = price-index без дивид. (BH занижен ~2%/г); это
  льстит разрыву, но maxDD-резка и по-декадная робастность —
  реальны и инвариантны к дивидендам.
- Фундаментал индекса = CAPE → 10-летние доходности, НЕ
  часы/дни (Campbell-Shiller). Короткого сигнала не даёт.
- «Часы» — нет данных, нет внутридневного edge.
- Торговля — через TradFi-перп BingX (NDX/SP500 в юниверсе
  юзера): funding/гэпы/часы → ОТДЕЛЬНЫЙ план риска+demo, НЕ
  деплой по этому бэктесту. Live — правила CLAUDE.md в силе.

**Статус:** реальный робастный кандидат (Faber на индексе).
Следующий честный шаг — отдельный план: перп-имплементация
(издержки/funding/гэпы перпа vs индекса), затем demo ≥4 нед,
затем разговор о деньгах. НЕ p-hacking — канон, не скан.

## 37.4 КРОСС-АССЕТ ROBUSTNESS (2026-05-17) — тезис подтверждён

`scripts/faber_cross_asset.py`: та же фикс-Faber200 (не подгон)
на 11 инструментах списка владельца с глубокой Yahoo-историей.

Faber улучшил risk-adj у **9–11/11**: Gold Sh0.63→1.21,
Copper 0.33→1.10, Oil →1.16 (BH-число битое = реальная
отрицательная цена нефти апр-2020, Faber её ОБОШЁЛ — плюс TF),
Dow 0.46→1.53, USDJPY 0.08→1.07, Apple 0.42→1.22, NVDA
0.50→1.27, Tesla 0.64→1.38, GOOGL 0.65→1.35, Meta 0.64→1.41,
MSTR 0.13→0.95. Везде ~×2 Sharpe и ~½ maxDD.

**Вывод:** трендследование — РОБАСТНЫЙ кросс-ассет феномен
(managed-futures тезис), подтверждён на индексах + металлах +
нефти + FX + мегакап-акциях, десятилетия, предзадано, без
подгона. Контраст с крипто (планы 31–36, edge нет): у этих
активов есть фундамент (премия за риск/тренды). Это и есть
честно найденная «работающая в мире» стратегия.

Честно: единичные акции шумнее (2 «—» — там Faber всё равно
резко лучше по Sharpe, флаг по строгому DD-порогу); это
оценка ШИРОТЫ edge, НЕ деплой. Путь к деньгам — план 38
(перп BingX + funding/гэпы) → demo ≥4 нед → явное «да».
Крипта не дублируется (план 36: TSMOM-крипта не прошла).
