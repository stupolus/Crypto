# План 45 — GTAA-портфель на BingX VST (маппинг + жизнеспособность)

## Контекст

План 43 подтвердил Faber-GTAA на ETF/индексах (^GSPC, ^NDX,
GC=F, CL=F, IEF). Для переноса на demo нужно решить базовый
вопрос: **какие из 5 активов есть как перпы на BingX VST**,
и достаточно ли их для воспроизведения сути идеи (диверсификация
несколько классов через 200-SMA триггер).

Из прошлой работы (план 41) знаю: BingX VST имеет
`NCSINASDAQ1002USD-USDT` (NASDAQ100-перп). Остальное —
нужно сканировать.

## Жёсткое предусловие

Канон Faber-GTAA — это 5 НЕКОРРЕЛИРОВАННЫХ классов:
US-large + US-tech + gold + oil + bonds. Если на VST не
найдётся ≥4 классов (любых, не обязательно ровно эти тикеры —
но из разных асс-классов), **диверсификация не работает** и
перенос не имеет смысла. Тогда:
- Faber-single ^NDX (план 41) — единственная live-форма GTAA-
  идеи на этой бирже.
- GTAA остаётся валидным бэктест-результатом без deploy.

Это не подгонка — это честное «инструмент не позволяет».

## Фазы

  - Если ≥4 классов есть → план 46 «GTAA → VST конфигурация

  - Если <4 → закрыть план честно: GTAA-deploy невозможен на
    VST, Faber-single ^NDX остаётся.

## 45.2/45.3 РЕЗУЛЬТАТ (2026-05-19) — 4/5 классов доступны

Скан 710 контрактов `swap/v2/quote/contracts`, фильтр на
TradFi-тикеры (NCSI/NCCO/NCFX/SPX/XAUT):

| GTAA актив | BingX VST перп |
|---|---|
| `^GSPC` US-large | `NCSISP5002USD-USDT` (или 724-вариант для 24/7) ✓ |
| `^NDX` US-tech | `NCSINASDAQ1002USD-USDT` (или 724) ✓ |
| `GC=F` Gold | `NCCOGOLD2USD-USDT` ✓ |
| `CL=F` WTI | `NCCO1OILWTI2USD-USDT` (или 724) ✓ |
| `IEF` 10y bonds | ❌ нет BOND/TREASURY/IEF/AGG/TLT-перпов на VST |

**Жизнеспособность**: 4-актив бэктест на тех же исторических
данных (^GSPC, ^NDX, GC=F, CL=F — без IEF) через ТОТ ЖЕ
строгий гейт плана 43:

| Метрика | 5-актив (план 43) | 4-актив (без IEF) | Гейт |
|---|---|---|---|
| OOS ann-Sharpe | 1.09 | **1.03** | >0.8 ✓ |
| OOS PF | 2.28 | 2.19 | >1.3 ✓ |
| OOS t-stat | 3.69 | 3.65 | >2 ✓ |
| MaxDD | −8.5% | −13.8% | < BH(−52.6%) ✓ |
| WF фолды | 4/4 | 4/4 | ≥3/4 ✓ |

**4-актив GTAA ПРОШЁЛ** — edge сохраняется, чуть слабее, но
в пределах строгого гейта. Bond-leg удалена не по подгонке, а
по отсутствию инструмента — это не cherry-picking, а
honest-instrument-constraint.

**Решение**: открываем план 46 — GTAA-4 на BingX VST demo.
Универсалия FIXED: 4 перпа выше, SMA=200 на daily, ежемесячный
ребаланс, equal-weight 1/4 ON / cash OFF, B-tier ≤3x.

Faber-single ^NDX (план 41) продолжает demo параллельно —
не отменяется. План 46 — отдельный systemd-юнит, отдельный
state-файл, отдельный лог.
