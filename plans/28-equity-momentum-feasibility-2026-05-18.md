# План 28 — equity momentum: feasibility-стоп по survivorship bias

## Дата: 2026-05-18 · База: план 27 + retro, retro 2026-05-18-equity-trend-wf

## Контекст

План 27: trend_ema на дневках даёт PF 2.23 (AAPL) / 1.78 (NVDA), но
это **selection/hindsight bias** — мы выбрали известных победителей.
Честная проверка требует point-in-time вселенной без знания будущих
победителей и **с делистнутыми проигравшими** (survivorship-free).

## Feasibility-проба (выполнена)

Yahoo Finance по делистнутым/банкротам **данных не отдаёт**:

| Тикер | Событие | Yahoo |
|---|---|---|
| LEH (Lehman) | банкрот 2008 | NO DATA |
| ENE (Enron) | банкрот 2001 | NO DATA |
| BSC (Bear Stearns) | поглощён 2008 | NO DATA |
| WAMUQ (WaMu) | банкрот 2008 | NO DATA |
| SIVBQ (SVB) | крах 2023 | Not Found |

**Вывод: survivorship-free вселенная на бесплатных данных невозможна.**
Любая momentum-стратегия на Yahoo-вселенной зашьёт ровно тот bias,
который мы вскрыли в плане 27 → результат методологически невалиден.
Строить такое — против дисциплины (никаких фейковых edge).

## Развилка (решение за пользователем)

- **A. Платный survivorship-free источник** (Norgate / Sharadar /
  EODHD-with-delisted / CRSP). Единственный научно валидный путь.
  Нужны: бюджет + ключи + решение по сетевой политике. Отдельный
  большой план (новый адаптер данных + cross-sectional momentum
  стратегия + WF).
- **B. Документированный bounded-bias тест.** Текущая ликвидная
  вселенная (нынешние S&P500/Nasdaq100) по истории. Bias известен и
  оптимистичен; результат — НЕ развёртываемый, только «работает ли
  механика в принципе» (а это план 27 уже показал на AAPL/NVDA).
  Низкая научная ценность, риск самообмана.
- **C. Стоп equity-ветки, возврат в крипту.** Там данные чистые и
  обильные (BingX), венчур исполнения родной. Equity-экскурс дал
  durable инфру (`download_equity`) и ясный негатив: ни btc_breakout,
  ни trend_ema на доступных данных не дают валидируемый edge.

## Рекомендация

**C (с опцией A, если пользователь готов финансировать данные).**
Гнать B — против принципа «не натягивать»: мы заранее знаем, что bias
сделает результат красивым и ложным. Честный итог equity-арки: без
платных survivorship-free данных edge на акциях не валидируется;
основная работа — крипта.

## Что НЕ делаем
Не пишем momentum-стратегию на Yahoo-вселенной. Это воспроизведёт
вскрытый bias и нарушит дисциплину. Код денежного пути не трогаем.
