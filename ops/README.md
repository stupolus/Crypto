# ops/

Эксплуатационные скрипты: подключение к VPS, tmux-сессии, деплой, мониторинг (позже).

## crypto-shell.sh

Подключение к постоянной tmux-сессии `crypto` в `/root/Crypto`. Сессия живёт в фоне на VPS — отключайся в любой момент, при следующем подключении продолжишь с того же места.

### Запуск с любого устройства

```bash
ssh root@187.124.41.13 -t /root/Crypto/ops/crypto-shell.sh
```

Флаг `-t` важен — он включает интерактивный режим, без него tmux не запустится.

### Удобный алиас на Mac/iMac (добавь в `~/.zshrc`)

```bash
alias crypto="ssh root@187.124.41.13 -t /root/Crypto/ops/crypto-shell.sh"
```

После этого с любого Mac пишешь `crypto` в Терминале — попадаешь в tmux-сессию.

### Управление tmux (минимум)

- **Отключиться (сессия продолжит жить):** `Ctrl+B`, потом `D`
- **Прокрутка истории:** `Ctrl+B`, потом `[`. Выход из режима: `q`.
- **Закрыть сессию совсем:** в shell внутри tmux напиши `exit`. Или снаружи: `tmux kill-session -t crypto`.

### Запуск Claude Code в этой сессии

Внутри tmux:
```bash
claude
```
Будет работать в `/root/Crypto`, видеть весь репо. Закрыл крышку MacBook — Claude продолжает крутиться на VPS. Открыл iPhone, зашёл через SSH в ту же tmux — видишь продолжение разговора.
