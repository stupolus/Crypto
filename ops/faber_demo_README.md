# Faber demo paper-runner — расписание (план 40.3)

PAPER-only форвардное demo (план 40). Без реальных ордеров и
торговых ключей. Валидирует перп-слой Faber-тайминга, который
бэктестом проверить нельзя (план 38.2: BingX-перп ~149д).

## Запуск вручную
```
.venv/bin/python -m scripts.faber_demo_runner
```
Идемпотентно: один день — одна строка в `ops/faber_demo.jsonl`.

## Ежедневное расписание (cron)
Раннер должен вызываться раз в сутки (после закрытия US-сессии,
напр. 22:00 UTC). Пример crontab:
```
0 22 * * *  /home/user/Crypto/scripts/faber_demo_cron.sh
```
(сделать исполняемым: `chmod +x scripts/faber_demo_cron.sh`)

## Вердикт
НЕ в сессии. Через ≥4 недели (лучше ≥8) — оценить по
ЗАФИКСИРОВАННОМУ в плане 40 критерию (tracking corr≥0.95 /
|базис|≤0.5%, funding-drag ≤⅓, гэпы в модели, синхрон сделок,
0 нарушений лимитов). Критерий НЕ двигать. Прошёл → отдельный
явный разговор о live по правилам CLAUDE.md. Не прошёл →
честно закрыть.
