# ВЕРДИКТ: GO / NO-GO по торговому эджу — 2026-05-19

## ИТОГ: **NO-GO** (на доступных бесплатных данных валидируемого эджа нет)

Планка приёмки заморожена ДО прогонов (план 49), снижена до
реалистичной пост-издержками (PF≥1.20, Sharpe≥0.8, net>0, OOS+≥55%,
≥30 сделок, DD≤35%, на BTC И ETH). Даже по этой **мягкой** планке —
ни одна стратегия не прошла.

| Гипотеза | n | Вердикт |

Итого: **≈6 стратегий-семей + 4 info-edge гипотезы (~37 стат-тестов)**.
Единственный «значимый» сигнал (BTC↔ETH) не пережил OOS — дисциплина
сработала, фейк не пропущен.

## Единственный незакрытый эксперимент

**composite_signal forward-демо** — крутится на VPS (бумага/VST, 4h),
kill-критерий заморожен (−5% / 5 убытков подряд / PF<0.8@30; success
PF≥1.5 & net>0 & ≥55% недель; горизонт ≥4 нед). Бэктест-свидетельство
по нему отрицательное → это **long-shot**, не ожидаемый GO. Решение по
нему — автоматически по kill/success через реальное время, БЕЗ моего
вмешательства. Это последний **бесплатный** шанс.

## Что РЕАЛЬНО могло бы изменить картину (и стоит ли)

Свободные данные (klines + Yahoo macro + DefiLlama on-chain) для
directional intraday/swing **исчерпаны** — результат структурный
(розничные данные/латентность ≈ efficient market). Чтобы найти эдж,
нужен один из путей, и **все упираются в деньги или иной подход**:

1. **CryptoQuant** (~$30–100/мес): биржевые потоки (BTC inflow/outflow,
   miner outflow) — единственный непроверенный класс «smart-money»
   данных с сильнейшим анекдотическим эджем. **Дешёвый тест:** $30 ×
   1 мес → pre-registered probe «exchange inflow → ΔBTC» (та же
   методика). Проверяет 1 гипотезу. Низкий риск денег, высокая
   информационная отдача.
2. **Coinglass upgrade** (~$30–700/мес): даёт intraday + ≥12мес →
   честный WF composite (сейчас невозможен). Проверяет: есть ли эдж
   у уже построенного композита.
3. **Другой класс активов** (акции): нужен survivorship-free источник
   (Norgate/Sharadar ~$30–150/мес) — без него equity-momentum невалиден.
4. **Другой подход** (не directional): market-making/ребейты,
   позиционные тренды на дневках с другим риск-профилем. Крупная
   переработка, не «ещё одна стратегия».

## Рекомендация (решение за владельцем — это деньги)

- **Бесплатно и сразу:** дождаться результата composite forward-демо
  (≥4 нед). Ничего не стоит, идёт само.
- **Если готов вложить минимум:** $30 на 1 мес CryptoQuant → один
  чистый pre-registered probe exchange-flow. Самый дешёвый способ
  проверить принципиально другой класс данных. Если и он null —
  сильный сигнал, что розничный крипто-directional-эдж недостижим на
  доступных средствах, и проект стоит переориентировать (другой
  подход) или закрыть.
- **Без вложений и без новых данных** дальнейший поиск даст тот же
  NO-GO — это уже доказано 10 гипотезами. Не рекомендую жечь время.

## Состояние

Прод/демо не тронуты. Всё через PR. Гейты зелёные (692 теста). Реальным
капиталом не рисковал, платных данных не покупал. Дисциплина «честный
NO-GO лучше натянутого плюса» соблюдена: ни один результат не натянут.
