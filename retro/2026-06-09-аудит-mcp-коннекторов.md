# Аудит скилов, плагинов и MCP-коннекторов — 2026-06-09

## Итог

- **Скилы и плагины проекта: чисто.** В репозитории и контейнере нет ни `.claude/skills/`, ни плагинов, ни кастомных слэш-команд. Доступные скилы (`code-review`, `verify`, `deep-research`, `security-review` и т.д.) — встроенные в Claude Code, весят мало, отключать нечего и не нужно.
- **Перегрузка идёт от MCP-коннекторов аккаунта claude.ai: подключено 27 серверов, ~400+ инструментов.** Для проекта (торговый бот) реально нужен **1**: GitHub. Описания всех серверов и имена всех инструментов инжектятся в контекст каждой сессии — это съедает контекстное окно ещё до начала работы (а правило проекта — «контекст ≤60%»).

## Как отключить

В **claude.ai → Settings → Connectors** — отключить ненужные глобально, либо снять галочки для конкретного окружения Claude Code on the web (настройки окружения → инструменты/коннекторы). Из самого репозитория это не настраивается — конфиг живёт на уровне аккаунта.

## Эффект

Отключение лишнего убирает из каждой сессии ~350 объявлений инструментов и несколько крупных блоков инструкций (Figma, Gamma, Lawvable, Apollo и др.) — меньше шума, больше контекста под реальную задачу, быстрее старт сессии.
