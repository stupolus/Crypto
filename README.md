# Crypto

Автоматизированная торговая платформа.

**Стартовая точка для агентов:** [`CLAUDE.md`](./CLAUDE.md).
**Стартовая точка для людей:** [`бизнес/INDEX.md`](./бизнес/INDEX.md).
**Текущий мастер-план:** [`plans/00-стратегия-проекта-2026-05-09.md`](./plans/00-стратегия-проекта-2026-05-09.md).

## Принципы

1. Никакого кода до плана.
2. Чужие сигналы — данные, не входы.
3. Размер позиции от риска, не от плеча.
4. Каждое решение — в файлы.
5. Никогда плечо > 5x.

## Структура

- `бизнес/` — второй мозг (цели, риск-профиль, материалы, идеи).
- `plans/` — техпланы фич.
- `retro/` — ретроспективы.
- `parsers/` — внешние источники данных.
- `adapters/` — интеграции с биржами.
- `core/` — общая инфраструктура (риск, сигналы, бэктест).
- `strategies/` — торговые стратегии.
- `data/` — собираемые данные (gitignored).
- `журнал/` — журнал сделок и review.
- `ops/` — мониторинг и эксплуатация.
